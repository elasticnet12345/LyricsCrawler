# LyricsCrawler

 ### 歌詞サイト
- うたまっぷ
- 歌詞タイム
- J-Lyric.net
  
現在, 対応はJLyric.netのみ. 最終的には全て対応させるつもり.

### 該当URL
タブ関係は全て同じだが, 歌手と曲の歌詞のページの階層関係が明確なのでJ-Lyric.netがオヌヌメ

### 使い方
カレントディレクトリに, 「水樹奈々_Astrogation_歌詞.txt」というファイルができる.
```shell
$ python crawler.py http://j-lyric.net/artist/a0018f8/l00c342.html .
```
